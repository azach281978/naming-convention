
# Azure Naming Generator

This tool helps users easily identify and apply the standard Azure naming convention, removing guesswork and eliminating mistakes when creating resource names.

The Azure Naming Generator is a lightweight, browser-based tool designed to help teams create Azure resource names that fully comply with the official naming standards. 

The tool aligns with the Microsoft Cloud Adoption Framework (CAF) and Azure Enterprise-Scale Landing Zone conventions to ensure consistent, predictable, and governance-ready naming across all workloads and environments.
